HeartGuard – Power BI Dashboard
Project Description

HeartGuard is a data visualization project developed using Power BI to analyze and interpret heart disease data. The dashboard presents key clinical insights such as survival rates, patient health indicators, and relationships between various medical attributes.

The primary objective of this project is to convert raw healthcare data into structured, meaningful insights that support better understanding of heart disease patterns and contributing risk factors.

Objectives
Analyze the heart disease dataset using visual analytics
Identify patterns in survival and mortality
Examine the impact of clinical features on heart health
Develop an interactive dashboard for efficient data exploration
Key Features
Survival rate analysis (survived vs deceased)
Visualization of patient health metrics such as age, cholesterol, and blood pressure
Trend analysis between clinical variables and outcomes
Interactive filters (slicers) for dynamic exploration
Key performance indicators (KPIs) for summary insights
Tools and Technologies
Power BI
CSV dataset (Heart Disease Dataset)
Dataset Information

The dataset includes the following attributes:

The dataset contains clinical and demographic attributes of patients used to analyze heart disease outcomes. It includes variables such as age, gender, and medical conditions like anaemia, diabetes, and high blood pressure. Additionally, it captures important physiological measurements including creatinine phosphokinase levels, ejection fraction, platelet count, serum creatinine, and serum sodium.
Lifestyle factors such as smoking status are also included, along with a follow-up period indicating the duration of observation. The dataset is structured to support outcome analysis, where the target variable (DEATH_EVENT) represents whether the patient survived or not.
These attributes collectively enable a comprehensive analysis of risk factors, patient health conditions, and their impact on survival outcomes in heart disease cases.
